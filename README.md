# MemoChat

该项目基于 C++26 与 Qt/Qml6.8.2 为基准的实验性平台，主要测试 Glaze 反射 stdexec 协程库 QUIC HTTP3等，不具备长期维护性，且该项目内容较繁杂，多个无关功能堆叠，但后端以具备微服务骨架，能较好拆分模块，未来可能会直接从该项目直接抽出某一模块功能进行长期维护。

目前该项目有正常 IM 服务，Live2D AI 化操纵，通过 OpenCV 与 GPT-SoVIST 进行语音模拟，接入 Agent 进行智能操作，未来可能接入直播平台进行无人直播。项目内有 R18 分区，通过接入各个网站爬取媒体源，主要实现参考 Venera 这个项目。
