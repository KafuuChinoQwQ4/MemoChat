Qdrant collections

RAG vector store
user_{uid}_{kb_id}
每个用户知识库一个 collection；当前只读查询到 104 个 collection。
vector
向量维度来自 embedder，默认配置 384；距离度量为 COSINE。
page_content
chunk 文本。
metadata
源文件、页码等扩展元数据。
kb_id / uid / chunk_index
知识库、用户和 chunk 序号。
source / file_type / page / total_chunks / content_length
检索、过滤和展示用 payload 字段。

Neo4j graph

AI graph memory
User
uid、updated_at；AI 图记忆的用户节点。
AgentSession
session_id、uid、updated_at。
AgentTurn
turn_id、uid、session_id、user_message、ai_message、created_at。
Topic
name、updated_at；从用户消息和 AI 回复中抽取。
User -[:HAS_AGENT_SESSION]-> AgentSession
用户到 AI 会话。
AgentSession -[:HAS_TURN]-> AgentTurn
会话到轮次。
User -[:INTERESTED_IN]-> Topic
兴趣权重随交互累加。
AgentTurn -[:ABOUT]-> Topic
轮次涉及的主题。
