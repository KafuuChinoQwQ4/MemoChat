# Harness Engineering Evolution

MemoChat AIOrchestrator evolves into a production Agent Harness that keeps the project lightweight and local-first while absorbing proven ideas from high-adoption agent frameworks.

The harness owns:
- explicit run graphs and state transitions
- model/provider routing
- tool registry and MCP execution
- RAG and graph memory
- guardrails and policy checks
- trace trees, replay, feedback, and regression evaluation

The design is framework-informed but not framework-bound. External frameworks are references for patterns; MemoChat keeps its own contracts, Docker topology, C++ AIServer compatibility, and QML trace/debug experience.
