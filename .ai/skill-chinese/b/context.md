# 上下文

## 任务

用户要求在 skill 中加入新规则：每次使用 agent 时遵循数据库 ACID 思路，包括原子性、一致性、隔离性和持久性。

## 相关文件

- `AGENTS.md`：仓库级 agent 指令，应包含全局规则。
- `skills/SKILL.md`：主 MemoChat 任务流水线，应包含所有任务都要遵守的规则。
- `skills/task.md`：常规实现工作流，适合补充具体执行约束。
- `skills/PROMPTS.md`：委托 agent 的共享规则模板，应提醒委托任务遵循 ACID。

## 数据和服务依赖

无。此次只修改文档规则，不触碰 Docker、数据库、运行时配置或业务代码。

## 安全边界

规则中的失败回滚/stash 必须只作用于本 agent 本轮产生的改动。当前仓库存在大量用户/其他任务未提交改动，因此本次不执行提交、stash 或回滚。
